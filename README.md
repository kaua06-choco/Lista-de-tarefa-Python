# 📝 Lista de Tarefas (To-Do List)

Projeto simples em Python para gerenciar tarefas pelo terminal.

## 🚀 Funcionalidades
- ➕ Adicionar tarefas
- 📋 Listar tarefas
- ❌ Remover tarefas

## 🧠 Tecnologias utilizadas
- Python 3

## ▶️ Como executar

1. Certifique-se de ter o Python instalado:
   python --version

2. Execute o arquivo:
   python nome_do_arquivo.py
